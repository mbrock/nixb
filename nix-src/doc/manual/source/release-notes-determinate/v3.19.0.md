# Release 3.19.0 (2026-04-28)

* Based on [upstream Nix 2.34.6](../release-notes/rl-2.34.md).

## What's Changed

### Update to upstream Nix 2.34

Determinate Nix is now based on [upstream Nix 2.34](../release-notes/rl-2.34.md), which includes a large number of bug fixes and improvements.

### Provenance is preserved when copying paths to the daemon

Previously, `nix copy` would not forward provenance information to the recipient.

This has been fixed, such that building a derivation on machine A and `nix copy`ing it to machine B will show the same provenance.

PR: [DeterminateSystems/nix-src#399](https://github.com/DeterminateSystems/nix-src/pull/399)
